    Model guide
    
          
    
            Pick a local LLM that fits your machine. All tiers work with
            Ollama — run ollama pull <model> then set it in
            the  tab.
          
    
        
    

        
    
          Privacy first.
          Local models keep every word on your machine — no accounts, no cloud costs, no data
          retention. Online providers (OpenAI, Anthropic, OpenRouter…) are convenient but send your
          voice transcriptions and prompts to external servers. For a voice coding tool, we
          strongly recommend a local model whenever your hardware supports it.

          Not sure what tier you are? Quick check
          
    
            RAM: Windows → Task Manager → Performance → Memory.
            macOS → Apple menu → About This Mac → Memory.
            Linux → free -h
    
            GPU VRAM: Windows → Task Manager → Performance → GPU.
            macOS → About This Mac → Graphics. Linux → nvidia-smi or rocm-smi
    
            Rule of thumb: the model's file size must fit in VRAM for GPU inference,
            or in RAM for CPU inference (leave ~2 GB headroom for the OS).
